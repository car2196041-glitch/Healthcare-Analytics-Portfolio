# Revenue Cycle KPI Analyzer (C++)

Mock Organization: Northstar Health Partners  

## Overview
This C++ application analyzes healthcare revenue cycle data and calculates key KPIs:
- Denial Rate  
- Collection Rate  
- Outstanding Balance  

## Technologies
- C++
- CSV file processing
- Object-Oriented Programming

## Files
- main.cpp
- sample_accounts.csv

## Business Value
Demonstrates how healthcare organizations monitor revenue performance and identify financial risks.
